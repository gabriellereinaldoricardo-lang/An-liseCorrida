# Análise Corrida

📋 Sobre o Projeto

Dashboard interativo desenvolvido no Power BI para análise de performance em corridas. O projeto permite acompanhar evolução de treinos por meio de indicadores como distância, tempo e pace, auxiliando na tomada de decisão e melhoria de performance esportiva.

💡 Benefícios do projeto

* 📈 Acompanhamento da evolução de performance ao longo do tempo
* ⚡ Identificação de melhora ou queda no pace
* 🏃‍♀️ Controle do volume de treinos (distância e frequência)
* 🎯 Apoio na definição de metas de corrida
* 📊 Visualização clara e rápida dos resultados sem precisar analisar planilhas complexas
* 🔍 Facilidade para identificar padrões de desempenho
